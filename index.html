project aim

This project aims to visualize the individual or collective movememts among the animals(sheeps).
Background
Biologists requrie to understand the activities peformed by the animals to better study the animal behaviour is various cases.
Computer Scientist help biologist to collect the neccessary information from the animals by providing them with a bracelet and collar belt which include various devices.
This project helps understand the collected readings (magnetometer, location, activities) better.

Some Main Features

Data collected:
Location,Accelerometer,Gyroscope,Magnetometer readings
Activities performed in certain time intervals

Features included in the visualization: 3D model of animals
Bracelet and collar belt which change colour according to the activity change
Path of the sheeps coloured coded with respect to sheep id
Sheep's facing direction can be judged from the triangle visible in front of the sheep
Activty chart and accelerometer chart to keep take of readings
Path Chart to keep track of the location of the sheeps
